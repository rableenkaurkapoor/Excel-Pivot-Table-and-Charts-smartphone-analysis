# Smartphone Sales Trend Analysis using Excel

## Project Overview

In this project, I analyzed smartphone sales data using Excel by applying PivotTables and PivotCharts to transform raw transactional data into meaningful insights. The analysis focuses on identifying sales patterns across annual, quarterly, and monthly time levels to support data-driven decision making.

## Key Skills Demonstrated

- Data Analysis  
- Data Aggregation and Summarization  
- Time Series Analysis  
- Trend Identification  
- PivotTable Design and Optimization  
- PivotChart Visualization  
- Business Insight Generation  
- Data Interpretation and Reporting   

## Tools and Techniques

- Microsoft Excel  
- PivotTables  
- PivotCharts  
- Line Charts  
- Date Grouping (Year, Quarter, Month)  
- Data Structuring and Transformation  

## Analysis Approach

The dataset was structured and converted into a tabular format, followed by the creation of PivotTables to aggregate sales data. Date fields were grouped into Year, Quarter, and Month levels to enable multi-level time series analysis. PivotCharts were then used to visualize trends and support interpretation.

This analysis demonstrates how raw sales data can be transformed into structured insights using Excel. The ability to analyze trends across multiple time dimensions is critical for forecasting, performance tracking, and strategic decision making in business environments.
